## 🎯 API 端点清单

### 认证 API
- ✅ `POST /api/auth/register` - 用户注册
- ✅ `POST /api/auth/login` - 用户登录
- ✅ `GET /api/auth/profile` - 获取用户信息（需要认证）
- ✅ `PUT /api/auth/profile` - 更新用户信息（需要认证）

### 工具 API
- ✅ `GET /api/tools` - 获取用户的所有工具（需要认证）
- ✅ `POST /api/tools` - 创建新工具（需要认证）
- ✅ `PUT /api/tools/:id` - 更新工具（需要认证）
- ✅ `DELETE /api/tools/:id` - 删除工具（需要认证）

### 健康检查
- ✅ `GET /api/health` - 服务器健康状态检查

## 📊 数据模型

### User（用户）
```typescript
{
  username: string;      // 用户名，唯一，最少3个字符
  password: string;      // 加密后的密码
  nickname: string;      // 昵称
  createdAt: Date;       // 创建时间
}
```

### Tool（工具）
```typescript
{
  name: string;          // 工具名称
  description: string;   // 工具描述
  tags: string[];        // 标签数组
  icon: string;          // 图标（emoji）
  userId: ObjectId;      // 所属用户ID
  createdAt: Date;       // 创建时间
}
```
