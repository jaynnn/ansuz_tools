# Ansuz Tools - 界面预览

本文档展示了应用的主要界面和功能。

## 1. 登录界面

**路径:** `/login`

**功能:**
- 用户名和密码输入
- 表单验证
- 错误提示
- 注册链接

**界面元素:**
```
+----------------------------------+
|        登录                      |
|                                  |
|  用户名                          |
|  [___________________________]   |
|                                  |
|  密码                            |
|  [___________________________]   |
|                                  |
|  [      登录      ]              |
|                                  |
|  还没有账号？ 注册               |
+----------------------------------+
```

**路径:** `/register`

## 6. 主题切换

**功能:**
- 深色/浅色模式切换
- 主题持久化

**颜色方案:**

### 浅色主题（默认）
- 背景色：白色 (#ffffff)
- 次要背景：浅灰 (#f5f5f5)
- 文字主色：深黑 (#1a1a1a)
- 文字次色：灰色 (#666666)
- 主题色：靛蓝 (#4f46e5)

### 深色主题
- 背景色：深黑 (#1a1a1a)
- 次要背景：更深黑 (#0f0f0f)
- 文字主色：白色 (#ffffff)
- 文字次色：浅灰 (#a0a0a0)
- 主题色：浅靛蓝 (#6366f1)

## 7. 响应式设计

**桌面版 (>768px):**
- 工具卡片：3-4列网格
- 侧边距：合适的留白
- 卡片宽度：最小 300px

**移动版 (<768px):**
- 工具卡片：1-2列网格
- 自适应布局
- 触摸友好的按钮大小

## 8. 交互效果

**鼠标悬停:**
- 卡片：上浮效果 + 阴影加深
- 按钮：颜色变化
- 链接：下划线显示

**焦点状态:**
- 输入框：边框颜色变为主题色
- 按钮：轮廓显示

**加载状态:**
- 登录/注册：按钮显示"加载中..."
- 数据加载：显示加载文字

## 总结

应用界面简洁、现代，注重用户体验：
- 清晰的信息层级
- 一致的视觉设计
- 流畅的交互动画
- 完善的主题支持
- 良好的响应式布局
